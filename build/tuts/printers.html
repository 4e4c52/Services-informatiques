Utilisation des Imprimantes Réseau

Découvrez comment utiliser les imprimantes réseaux de l'école.

  • Localisez et ajoutez l'imprimante

1 Informations Générales

L'imprimante réseau se situe dans la salle 305 au troisième étage de l'école.
Voici les caractéristiques de l'imprimante :

  • Nom de l'imprimante : BROTHER 5250DN
  • Adresse IP : 172.17.2.1
  • Type de connexion : port TCP/IP standard

2 Ajouter l'Imprimante

2.1 Sur Windows 7

Pour ajouter l'imprimante sur Windows 7, ouvrez le menu Démarrer puis allez dans Panneau de configuration où vous vous renderez dans Périphériques et imprimantes.

En haut de la page, cliquez sur le bouton Ajouter une imprimante.

Dans la fenêtre qui vient de s'ouvrir, cliquez sur Ajouter une imprimante réseau, sans fil ou Bluetooth.

Windows va à présent rechercher les imprimantes disponibles sur le réseau. Arrêtez la recherche de l'imprimante.

Cliquez à présent sur L'imprimante que je cherche n'est pas répertoriée puis cliquez sur suivant.

Dans la fenêtre qui s'affiche, choisissez Ajoutez une imprimante à l'aide d'une adresse TCP/IP ou d'un nom d'hôte.

Ensuite, entrez l'adresse IP de l'imprimante comme suit :

Vous allez à présent devoir installer le pilote de l'imprimante afin de pouvoir l'utiliser. Pour ce faire, téléchargez le pilote correspondant à votre ordinateur ci-dessous :

  • Pilote Windows 7 64 bits
  • Pilote Windows 7 32 bits

Une fois le pilote téléchargé, ouvrez l'archive puis dézippez son contenu sur votre bureau.
Cliquez maintenant sur Disque fourni.

Dans la fenêtre qui vient de s'ouvrir, cliquez sur Parcourir puis rentrez dans le dossier que vous venez de dézipper sur votre bureau. Vous ne devriez voir qu'un seul fichier.

Cliquez sur Ouvrir puis Ok.
Le contenu de la fenêtre devrait changer pour vous afficher une liste de quelques imprimantes. Choisissez BROTHER HL-5250DN series.

Cliquez sur suivant. Windows va maintenant installer l'imprimante.

Une fois l'installation terminé, donnez un nom à l'imprimante puis cliquez sur suivant. La fenêtre suivante vous propose de partager l'imprimante, refusez.

Cliquez une dernière fois sur suivant et c'est terminé !
Il n'est vraiment pas nécessaire d'imprimer une page de test.

3 Imprimez vos Fichiers

Pour imprimer un fichier, rien ne change, sélectionnez juste l'imprimante réseau qui portera le nom que vous lui avez donné lors de l'installation.
