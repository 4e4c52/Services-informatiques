Configuration et Utilisation de VirtualBox

Découvrez comment configurer et utiliser VirtualBox.

  • Téléchargez VirtualBox
  • Créez votre machine virtuelle

Chapitres

  2. Créer une Machine Virtuelle

1 Télécharger VirtualBox

VirtualBox est un logiciel de virtualisation permettant d'avoir plusieurs systèmes d'exploitation sur la même machine et fonctionnant au même moment. Ce logiciel est gratuit et est disponible en téléchargement sur le site officiel.

2 Créer une Machine Virtuelle

2.1 Télécharger une Image

Avant d'aller plus moin, vous devez avoir sur votre ordinateur une image disque d'un système d'exploitation. Ces fichiers prennent l'extension .iso et sont téléchargeables sur les sites des différentes distributions.

Vous pouvez télécharger CentOS ainsi que Ubuntu depuis les liens suivants :

  • CentOS
  • Ubuntu

Pensez bien à télécharger le fichier ISO prévu pour l'architecture de votre processeur (x86 ou x64).

2.2 Configuration de la VM

Après avoir téléchargé une image, sauvegardez là dans un dossier où vous pourrez la retrouver. Ouvrez ensuite VirtualBox et cliquez sur le bouton Créer.

Cliquez ensuite sur suivant. VirtualBox vous demande maintenant des informations sur la machine virtuelle que vous souhaitez créer. Vous devez dans un premier temps lui donner un nom (unique sur votre ordinateur) qui sera utilisé pour identifier la machine dans VirtualBox mais qui servira également de nom pour le fichier contenant votre machine virtuelle sur votre ordinateur. Après avoir nommé votre machine virtuelle, vous devez sélectionner son type à l'aide des listes déroulantes.

Cliquez sur suivant. Vous devez maintenant entrer le nombre de Megaoctets de mémoire vive que votre machine virtuelle est autorisée à exploiter. VirtualBox recommande 512 Mo mais 1 Go (1024 Mo) est tout de même préférable pour un confort d'utilisation optimal.

L'étape suivante traite du disque dur virtuel utilisé par VirtualBox pour sauvegarder votre machine. Si vous suivez ce tutoriel, laissez coché Disque dur d'amorçage et sélectionnez Créer un nouveau disque dur.

En revanche, si vous avez déjà un disque de machine virtuelle (au format .vdi pour VirtualBox), vous pouvez l'utiliser en sélectionnant Utiliser un disque dur existant et en allant le chercher dans vos dossiers.

Le fait de cliquer sur Suivant va ouvrir une nouvelle fenêtre permettant de créer le nouveau disque dur virtuel.

Cliquez sur Suivant et dans Type de disque virtuel, sélectionnez Fichier de taille variable. Ainsi, au fur et à mesure que votre machine virtuelle prendra de la place, le fichier la contenant s'agrandira. Plutôt sympa !

La fenêtre suivante pour demande de donner un nom à votre disque dur virtuel. Par défaut, le nom que vous avez donné à votre machine virtuelle est utilisé mais libre à vous de le changer si bon vous semble. Ce qui est important sur cette fenêtre est à la taille de base de votre disque dur virtuel. Cette taille doit être au minimum de 8 Go. J'ai pour habitude de mettre au minimum 20 Go pour avoir une bonne marge de manoeuvre.

Cliquez maintenant sur Terminer pour fermer la fenêtre de création du disque virtuel puis encore sur Terminer pour fermer la fenêtre de création de la machine virtuelle.

Notre machine virtuelle apparaît maintenant dans la colonne de gauche sur VirtualBox. Sélectionnez la pour avoir quelques informations sur elle.

2.3 Paramètres Hardware

Telle quelle, notre machine virtuelle nous est parfaitement inutile. Nous allons devoir configurer la partie hardware de cette dernière afin de la rendre utilisable.

Sélectionnez la machine virtuelle et cliquez sur Configuration. La fenêtre ci-dessous va s'ouvrir.

Par défaut, notre machine virtuelle n'utilisera qu'un seul coeur du processeur quand bien même il en comporte plusieurs. Si vous souhaitez que votre machine virtuelle utilise 2 coeurs ou plus, allez dans la partie Système puis cochez la case Activier les IO-APIC dans l'onglet Carte mère.

Notez la partie Ordre d'amorçage. La machine virtuelle tentera dans un premier temps de démarrer sur une Disquette puis sur un CD/DVD et enfin sur le disque dur virtuel.

Toujours dans la partie Système, allez dans l'onglet Processeur puis sélectionnez le nombre de coeurs que vous souhaitez utiliser.

Rendez vous maintenant dans la partie Affichage. Attribuez à votre machine virtuelle au minimum 25 Mo de mémoire vidéo (21 Mo sont nécessaires pour lire des vidéos HD) puis cochez la case Activer l'acclération 3D. L'accélération 2D n'est supportée que vous les machine virtuelles Windows.

Nous allons maintenant nous attaquer à l'étape essentielle du paramètrage de la machine virtuelle. A l'heure actuelle, la machine virtuelle n'est qu'un ordinateur sans système d'exploitation. Autrement dit, une coquille vide. Pour installer un système d'exploitation, il faut insérer le DVD d'installation dans l'ordinateur et démarrer ce dernier sur le DVD. C'est exactement ce que nous allons faire ici. Souvenez vous de l'ordre d'amorçage dont nous avons parlé un peu plus haut. La machine virtuelle tentera de démarrer sur le lecteur de CD/DVD. Nous allons donc devoir insérer notre DVD d'installation dans ce lecteur.

Pour cela, rendez vous dans la partie Stockage. Sur la gauche, vous pouvez voir l'arborescence de stockage contenant un contrôleur IDE (notre lecteur de CD/DVD) ainsi qu'un contrôleur SATA (notre disque dur virtuel). Sous le contrôleur IDE, vous pouvez voir une icône de disque avec vide à côté. Cliquez sur cette icône. La partie droite de la fenêtre intitulée Attributs va se mettre à jour. La ligne lecteur CD/DVD contient un autre icône de disque. Cliquez dessus et sélectionnez Choisissez une fichier de CD/DVD virtuel...

Allez chercher le fichier .iso que vous avez téléchargé tout à l'heure puis cliquez sur Ouvrir.

Cliquez sur OK pour sauvegarder et fermer la fenêtre des paramètres.

Vous pouvez à présent sélectionner votre machione virtuelle et cliquer sur Démarrer. Vous vous retrouverez alors avec une fenêtre d'installation de votre système d'exploitation.

N'oubliez pas de supprimer le DVD du lecteur de la machine virtuelle après l'installation sans quoi, elle démarrera toujours sur le DVD d'installation.

Vous avez à présent une machine virtuelle fonctionnelle. Libre à vous de la configurer par la suite et d'installer les Addons proposés par VirtualBox pour un meilleur confort d'utilisation.
