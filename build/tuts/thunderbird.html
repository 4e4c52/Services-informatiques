Configurer Thunderbird

Découvrez comment configurer Thunderbird pour accéder à vos emails.

  • Téléchargez Thunderbird
  • Configurez votre compte de messagerie

Chapitres

  2. Configurer Thunderbird

1 Télécharger Thunderbird

Avant de commencer, assurez vous d'avoir la dernière version de Thunderbird installée sur votre ordinateur. Si ce n'est pas le cas, rendez vous sur la page de téléchargement de Thunderbird puis téléchargez la dernière version. Une fois Thunderbird téléchargé, installez le en laissant les options par défaut (standard).

2 Configurer Thunderbird

2.1 Nouveau Compte

Une fois Thunderbird téléchargé et installé, démarrez le. Si c'est la première fois, Thunderbird vous demandera votre identité, dans ce cas, passez directement à la partie 2.2. Sinon, allez dans le menu Fichier > Nouveau > Comptes courrier comme sur l'image ci-dessous.

2.2 Identité

Thunderbird vous demande à présent votre identité. Entrez les informations demandées comme ci-dessous.

Une fois les informations données, cliquez sur le bonton Continuer.

2.3 Paramètres du Compte

Thunderbird devrait à présent tenter de trouver automatiquement les paramètres de votre compte. En vain. Modifiez les paramètres avec ceux donnés ci-dessous. Vérifiez que vous avez entré toutes les informations comme sur l'image un peu plus bas.

  • Identifiant : votre login ISEN
  • Serveur entrant : mail-brest.isen.fr
    • Protocol : POP
    • Port : 995
    • Type de sécurité : SSL/TLS
    • Authentification : Autodétection
  • Serveur sortant : smtp-brest.isen.fr
    • Protocol : SMTP
    • Port : 465
    • Type de sécurité : SSL/TLS
    • Authentification : Autodétection

Cliquez à présent sur Re-tester afin que Thunderbird vérifie et enregistre les paramètres.
Il est probable qu'une fenêtre s'ouvre comme sur l'image ci-dessous. C'est tout à fait normal. Cette fenêtre vous indique que la connexion est sécurisée mais que l'identité du site n'a pas été vérifiée par une autorité digne de confiance.
Cochez la case Conserver cette exception de façon permanente puis cliquez sur Confirmer l'exception de sécurité.

Thunderbird devrait à présent vous afficher quelque chose comme l'image ci-dessous à la place des paramètres de votre compte. Si vous avez fait une erreur en tapant les paramètres de votre compte, modifiez les paramètres et recommencez l'opération.

Enfin, vous pouvez cliquez sur le bouton Créer le compte.
Une fois encore, il est possible que vous receviez une alerte de sécurité. C'est plus ou moins la même que la précédente. Cochez Je comprends les risques puis cliquez sur le bouton Créer le compte.
Votre compte est maintenant sauvegardé et devrait apparaître dans la colonne à gauche du logiciel.

2.4 Mots de Passe

Il est possible que lorsque vous enverrez un email pour la première fois, Thunderbird vous redemande votre mot de passe. Si tel est le cas, entrez le et cochez la case Retenir le mot de passe.
