Configuration de l'Agenda sur l'ENT

Découvrez comment configurer votre agenda sur l'ENT.

  • Configurez votre agenda

Chapitres

  1. Configurer l'Agenda

1 Configurer l'Agenda

L'ENT vous permet d'accéder à votre emploi du temps.

Pour cela, rendez vous dans la rubrique Etudes > Agenda comme ci-dessous.

Allez ensuite dans Réglages > Calques.

Si les calques sont désactivés, cliquez sur Activer les calques.

Vous devriez maintenant avoir le message Les calques sont actuellement activés.

A présent, cliquez sur Ajouter un calque.

Dans Source, sélectionnez votre classe/section et sauvegardez.

Si vous rencontrez un problème avec les horaires, allez dans Réglages > Préférences puis choississez Europe/Paris comme fuseau horaire puis sauvegardez les préférences.

Votre agenda est à présent configuré et prêt à être utilisé.

Si vous utilisez un logiciel d’agenda, vous pouvez consulter ce document PDF pour plus d'informations.
