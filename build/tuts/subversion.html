Configuration et Utilisation de SVN

Découvrez comment configurer et utiliser le serveur Subversion de l'école.

  • Téléchargez TortoiseSVN
  • Configurez TortoiseSVN

Chapitres

  1. Télécharger TortoiseSVN
  2. Configurer Tortoise SVN

1 Télécharger TortoiseSVN

TortoiseSVN est un logiciel utilisé sur Windows pour communiquer avec un serveur Subversion. Vous pouvez le télécharger gratuitement depuis le site officiel.

2 Configurer TortoiseSVN

Avant de configurer TortoiseSVN, vous devez faire une demande de répertoire Subversion auprès des services informatiques de l'école. Vous pouvez faire cela depuis la page de demande de répertoire.

Une fois votre demande de répertoire reçu et traitée par l'école, vous recevrez un email vous informant comment configurer TortoiseSVN avec le répertoire SVN créé à cette occasion.
