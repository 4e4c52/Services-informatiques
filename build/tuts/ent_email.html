Configuration de la Messagerie sur l'ENT

Découvrez comment configurer votre messagerie sur l'ENT.

  • Connectez vous
  • Configurez votre compte
  • Gérez vos informations

Chapitres

  1. Connexion
  2. Configuration
  3. Redirection
  4. Absences
  5. Modification du Mot de Passe
  6. Filtrage des Emails

1 Connexion

Tout d'abord, rendez-vous sur le site de l'ENT. Cliquez maintenant sur Se connecter en haut à droite. Entrez votre identifiant ISEN et votre mot de passe (par défaut: isen29 ).
Pour rappel, l'identifiant ISEN est composé de façon suivante:

  • Première lettre de votre prénom
  • 5 premières lettres de votre nom
  • 2 derniers chiffres de votre année de sortie
  • Par exemple, John Smith de la promotion 2015 aura pour identifiant jsmith15

A présent allez dans l'onglet Communication et cliquez sur Courriel. Vous êtes maintenant sur votre boite de reception.

2 Configuration

Avant tout, il faut modifier votre identité par défaut en remplaçant votrelogin@iseb.fr par prenom.nom@isen.fr.
Pour cela, cliquez sur option dans le menu à gauche. Dans Editer les options pour en haut à droite, choisissez Courrier.

Dans la catégorie Option générales, cliquez sur Données personnelles. Complétez de la manière suivante:

  • Profil par défaut : identité pas défaut
  • Sélectionnez le profil que vous désirez modifier : identité par défaut
  • Nom du compte : identité par défaut
  • Nom complet : rentrez votre nom complet
  • Votre adresse d'envoi De : prenom.nom@isen.fr

Normalement vous devriez obtenir ceci :

Cliquez ensuite sur enregister les options.

3 Redirection

Le service mail de l'ENT dispose d'une redirection des mails, pour cela cliquez sur Redirection dans le menu à gauche.

4 Absences

Sur votre page mail de l'ENT vous trouverez également l'option absences, cette dernière permet de renvoyer automatiquement une réponse aux mails que vous recevrez pendant une période durant laquelle vous serez dans l'incapacité de répondre par vous même. Pour l'utiliser cliquez sur Absence dans le menu à gauche.

5 Modification du Mot de Passe

Pour modifier votre mot de passe sur l'ENT allez dans l'onglet Mon compte puis cliquez sur "Mot de passe.

6 Filtrage des Emails

Le service mail de l'ENT dispose d'un filtrage des mails, pour cela cliquez sur Filtage dans le menu à gauche et configurez les différentes options comme bon vous semble.
